Firebase Setup Guide

To make the Google login work, you need to set up your own Firebase project:

  1. Go to Firebase Console and create a new project
  2. Add a web app to your project by clicking "Add app" and selecting the web icon
  3. Register your app with a name (e.g., "ShopEase")
  4. Copy the Firebase configuration that looks like this: 
    const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT_ID.appspot.com",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID"
};
  5. Open the file js/firebase-config.js and replace the example configuration with your own
  6. In the Firebase console, go to Authentication â†’ Sign-in methods and enable Google as a provider
  7. Add your domain (or localhost) to the Authorized domains list
  8. Go to Firestore Database and create a database in test mode
  9. Refresh this page and the Google login should work!
Note: The API key error "API key not valid" means Firebase has rejected the API key in the configuration. You must create your own Firebase project and use your own API key.
